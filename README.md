# formulario
Se debe crear una tabla *'formulario'* con los campos:
```
id INT()
nombre VARCHAR()
ciudad VARCHAR()
correo VARCHAR()
rut VARCHAR()
juegas VARCHAR()
nacimiento DATE()
porque VARCHAR()
```
